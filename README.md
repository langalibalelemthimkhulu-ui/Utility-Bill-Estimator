# Utility Bill Estimator (WinForms)

## Project Overview
This project is a Windows Forms application developed in C# for the PRLN201 (Programming Languages II) course. The application estimates a household electricity utility bill based on metric measurements.

## Functionality
The system allows users to:
- Enter electricity consumption in kilowatt-hours (kWh)
- Enter the cost per kilowatt-hour (R/kWh)
- Calculate and display the total amount payable
- Clear input fields and reset the calculation

The application demonstrates event-driven programming, input validation, and basic calculation logic using Windows Forms controls.

## Tools and Technologies
- C#
- Windows Forms (WinForms)
- Visual Studio
- GitHub for version control and collaboration

## Design Approach
The interface follows a clean, government-style layout using readable fonts, neutral colours, and a background panel image to enhance professionalism and usability.

## Collaboration
This repository is shared with group members to support collaborative development. GitHub is used to manage version control and allow multiple contributors to view and edit the application design and code.


## Disclaimer
This application is for educational and estimation purposes only and does not represent an official utility invoice.
